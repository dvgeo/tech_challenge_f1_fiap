API de Consultas - Dados Embrapa

Bem-vindo à API de consultas de dados vitivinícolas da Embrapa.

Esta API permite acessar informações de produção, processamento, comercialização, importação e exportação de dados disponíveis no portal da Embrapa.

Como usar a API

Para acessar os dados, você pode utilizar as seguintes rotas:

  • /producao - Dados de produção
  • /processamento - Dados de processamento
  • /comercializacao - Dados de comercialização
  • /importacao - Dados de importação
  • /exportacao - Dados de exportação

Exemplo de chamada via Python:

import requests

response = requests.get('http://localhost:8000/producao')
print(response.json())
